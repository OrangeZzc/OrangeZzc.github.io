2、如何创建一个Java程序

在左边的栏目里选择Package Explorer - - > 右击 - - > new Java project - - > 输入Project name - - > 点击next - - > Finish - - > 对src右击(src是存放源代码)新建一个包 - - > 输入包名(通常为com.②.③,其中“com.②”为域名的倒写，③可以写包名。如：com.baidu.包名) - - > Finish - - > 对刚才新建的包进行右击，新建一个类 - - > 输入类名 - - > Finish

3、如何导入已有的工程

File - - > Import - - > General - - > Existing Projects into Workspace - - > Next - - > 浏览(Browse….)，选择工程现在的位置 - - > 然后按下图勾上对应的选项 - - > 最后点击Finish - - > 至此导入完成。

​ 注意：如果没有勾上“Copy projects into workspace”，删除了该工程，那么就打不开该工程了。

4、如何改现有工程的名字

①、改eclipse里的工程名字

​ 选择需要改的工程 –> 右击 –> Refactor(即：重构) –> Rename

②、改准备导入eclipse的工程名字

​ 改不了

5、如何导入已有的源文件

​ 直接复制（ctrl + c）此源文件，直接在指定的工程包下粘贴（ctrl + v）即可。

6、如果导入源文件后打开是乱码

​ 将要导入的源文件复制到桌面上，用eclipse打开然后将其另存为UTF-8的编码形式。然后再将此文件复制（ctrl + c）到指定的工程包下粘贴（ctrl + v）即可。如果导入后还是报错，那看看此源程序的开头是否有加“Package 包名;”。最后刷新一下即可。

7、如何在Eclispe中查看Java类库源代码

​ 在代码中按住“ctrl”键，移动鼠标到想看的代码出，然后点击即可。如果想查找此源代码的某个内容，如：想看一下String类里的chartAt()方法，直接使用“ctrl + o”，最后再输入chartAt()，即可。

8、如何在编写的代码中显示此程序的相关信息

​ 设置完后，当想在代码中使用这些信息，只需要输出“/** */”然后再回车即可。

补充：

​ 1、新建的workspace与原有的workspace的设置是不一样的，因为两者的”.metadata”文件不一样。

9、Debug调试

①断点

​ 对你想查看的内容打上断点，当程序执行到该断点时，就会暂时暂停。

②如何运行Debug？

右击选择Debug as Java application即可。

③Eclipse中常用的调试按键

操作 作用
step into 跳入(f5) 进入当前行所调用的方法中
step over 跳过(f6) 执行完当前行的语句，进入下一行
step return 跳回(f7) 退出当前行所调用的方法
resume 恢复 进入下一个断点，如果没有就结束
Terminate 终止 停止调试
drop to frame 回到当前行所在方法的第一行
